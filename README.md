# japan-stock-portal

スマホ中心で使う、日本株投資用の個人ポータルサイトです。

## できること

- 銘柄コードを入力して主要サイトへのリンクを自動生成
- サイト上から銘柄を追加・編集・削除
- ミネルヴィニSEPAスクリーナー
- 日本株スイングスクリーニング支援ツール
- 投資メモをブラウザ内に保存
- TradingViewウィジェットを後から埋め込めるスペースを用意

## ファイル構成

```text
japan-stock-portal/
├── index.html   # 画面の骨組み
├── style.css    # デザイン
├── script.js    # 基本の銘柄管理、リンク生成、メモ保存
├── sepa.js      # SEPA採点、Trend Template判定、ランキング
├── swing.js     # 日本株スイングスクリーナー、CSV取込、プロンプト生成
└── README.md    # この説明書
```

## 日本株スイングスクリーナーの使い方

この機能は、全市場から自動抽出するものではありません。APIなしの無料静的サイトなので、株探・Yahoo!ファイナンス・TradingViewなどで見つけた候補銘柄を、手入力またはCSV貼り付けで登録して評価します。

### 手入力の方法

1. サイト上部の `スイング抽出` を押す
2. `候補銘柄を手入力` を開く
3. 銘柄コード、銘柄名、市場、株価、売買代金、業績、チャート分類などを入力
4. `候補を保存` を押す
5. ランキングカードにA/B/C/除外寄りの評価が表示されます

### CSV貼り付けの方法

`CSV貼り付けで一括登録` を開き、1行目に列名、2行目以降に銘柄データを貼り付けます。

対応列：

```text
code,name,market,price,changePercent,marketCap,turnover,turnoverRankCount,earningsSummary,salesGrowth,opProfitGrowth,epsGrowth,operatingMargin,roe,hasUpwardRevision,chartType,chartPattern,isMaAligned,hasVolumeIncrease,isBreakout,hasLongUpperWick,hasPostEarningsWeakness,flowStrength,flowContinuity,theme,themeLeadership,watchPoint,breakPoint,pullbackPoint,neckline,ma25,recentHigh,entryDecision
```

`true`、`あり`、`○`、`はい` はチェック項目として扱われます。

## スイング評価スコアの見方

100点満点で評価します。

- 80点以上：A
- 65〜79点：B
- 50〜64点：C
- 49点以下：除外寄り

評価の考え方：

- 流動性・資金流入：最大30点
- 業績：最大25点
- チャート：最大30点
- テーマ性・継続性：最大15点
- 減点項目：最大-15点

減点される例：

- 売買代金が小さい
- 資金流入の継続性が弱い
- 長い上ヒゲあり
- 決算後失速あり
- 業績が弱いのにテーマだけで買われている

## 自動まとめ

ランキング下部に以下を自動生成します。

- 今の市場で主役になっている銘柄群
- これから資金が本格流入しそうな銘柄群
- ブレイクの質が高い銘柄
- 初動として面白い大底反転銘柄
- 形は良いが、まだトリガー待ちの銘柄
- 明日以降の監視Top10

## ChatGPT分析用プロンプト生成

`ChatGPT分析プロンプトを生成` を押すと、登録済み候補銘柄の一覧を含む分析依頼文を作れます。

使い方：

1. 候補銘柄を登録
2. `ChatGPT分析プロンプトを生成` を押す
3. テキストエリアに表示された文章をコピー
4. ChatGPTに貼り付けて詳細分析を依頼

## SEPAスクリーナーの使い方

1. `銘柄追加・編集` を開く
2. 銘柄の基本情報を入力する
3. `SEPAテクニカル入力` に株価・移動平均・52週高値安値・RS・VCP評価を入力する
4. 必要なら `業績・テーマ評価` を開いて、EPS成長率、売上成長率、営業利益成長率、決算評価、テーマ性、需給評価を入力する
5. `銘柄を保存` を押す
6. `ミネルヴィニSEPAスクリーナー` にランキングと判定が表示されます

## データ保存について

銘柄データ、SEPA入力、スイング候補、投資メモは、入力したスマホやブラウザの `localStorage` に保存されます。

注意点：

- 他の端末には同期されません
- ブラウザのデータを削除するとメモも消える場合があります
- GitHub上に個別メモや価格情報が公開されるわけではありません
- APIなしなので、株価、移動平均、売買代金、業績情報は手入力またはCSV貼り付けです
