# japan-stock-portal

スマホ中心で使う、日本株投資用の個人ポータルサイトです。

## できること

- 銘柄コードを入力して主要サイトへのリンクを自動生成
  - 株探
  - Yahoo!ファイナンス
  - TradingView
  - TDnet
  - EDINET
- サイト上から銘柄を追加・編集・削除
- 銘柄ごとにリスク管理データを保存
  - ステータス：保有 / 監視 / 候補
  - エントリー価格
  - 損切りライン
  - 利確ライン
  - 決算日
  - 銘柄メモ
- ミネルヴィニSEPAスクリーナー
  - Trend Template判定
  - SEPAスコア100点満点
  - A/B/C/見送り分類
  - VCP評価
  - 損切り率、利確余地、R:R表示
  - 決算日接近アラート
  - SEPAスコア順ランキング
- 投資メモをブラウザ内に保存
- TradingViewウィジェットを後から埋め込めるスペースを用意

## GitHub Pagesで公開する手順

1. GitHubでこのリポジトリを開く
2. `Settings` を開く
3. 左メニューの `Pages` を開く
4. `Build and deployment` の `Source` を `Deploy from a branch` にする
5. `Branch` を `main` にする
6. フォルダは `/root` を選ぶ
7. `Save` を押す
8. 数十秒〜数分待つと、公開URLが表示されます

スマホで操作する場合は、GitHubアプリよりもChromeでGitHubを開き、ブラウザの「PC版サイト」を使うと設定しやすいです。

## 銘柄を追加・編集・削除する方法

サイト上の `銘柄追加・編集` から入力できます。

基本項目：

- 銘柄コード
- 銘柄名
- テーマ
- ステータス
- エントリー価格
- 損切りライン
- 利確ライン
- 決算日
- 銘柄メモ

既存銘柄を編集する場合は、銘柄カード下部の `編集` またはSEPAランキング内の `SEPA入力を編集` を押してください。
削除する場合は、銘柄カード下部の `削除` を押してください。

## SEPAスクリーナーの使い方

1. `銘柄追加・編集` を開く
2. 銘柄の基本情報を入力する
3. `SEPAテクニカル入力` に株価・移動平均・52週高値安値・RS・VCP評価を入力する
4. 必要なら `業績・テーマ評価` を開いて、EPS成長率、売上成長率、営業利益成長率、決算評価、テーマ性、需給評価を入力する
5. `銘柄を保存` を押す
6. `ミネルヴィニSEPAスクリーナー` にランキングと判定が表示されます

## SEPA入力項目の説明

### テクニカル項目

- 現在株価：最新の株価を手入力
- 50日移動平均：短中期トレンド確認用
- 150日移動平均：中期トレンド確認用
- 200日移動平均：長期トレンド確認用
- 52週高値：高値圏かどうかを判定
- 52週安値：安値から十分上昇しているかを判定
- RS評価 1〜100：相対的な強さ。70以上で加点、80以上なら強め
- 出来高評価 1〜5：ブレイク時や上昇時の出来高の強さ
- ベース形成評価 1〜5：ベースがきれいかどうか
- VCP収縮評価 1〜5：値幅・出来高の収縮が進んでいるか
- ピボット価格：ブレイク判断に使う価格
- ブレイク済み：ピボットを突破済みならオン

### 業績・テーマ項目

- EPS成長率：利益成長率
- 売上成長率：売上の伸び
- 営業利益成長率：本業利益の伸び
- 決算評価 1〜5：直近決算の質
- テーマ性 1〜5：資金が入りやすいテーマか
- 需給評価 1〜5：出来高、信用需給、上値の軽さなどの主観評価

## Trend Template判定

以下をチェックし、何個満たしているかを `Trend 7/8` のように表示します。

- 現在株価 > 50日移動平均
- 現在株価 > 150日移動平均
- 現在株価 > 200日移動平均
- 50日移動平均 > 150日移動平均
- 150日移動平均 > 200日移動平均
- 現在株価が52週安値より25%以上高い
- 現在株価が52週高値から25%以内
- RS評価が70以上

## SEPAスコアの見方

100点満点で以下のように分類します。

- 80点以上：Aランク SEPA候補
- 65〜79点：Bランク 監視継続
- 50〜64点：Cランク 形待ち
- 49点以下：見送り

配点の考え方：

- Trend Template適合度：30点
- 相対力・株価位置：15点
- VCP/ベース形成：20点
- 出来高・需給：10点
- 業績成長：15点
- テーマ性・決算評価：10点

## リスク管理表示

エントリー価格、損切りライン、利確ラインから以下を表示します。

- 損切り率
- 利確目標までの上昇率
- R:R
- 損切りが深い場合の警告
- 決算日が近い場合の注意表示

目安：

- 損切り率8%以内：良好
- 8〜12%：やや深い
- 12%超：警告
- R:R 1:2以上：良好

## データ保存について

銘柄データ、SEPA入力、投資メモは、入力したスマホやブラウザの `localStorage` に保存されます。

注意点：

- 他の端末には同期されません
- ブラウザのデータを削除するとメモも消える場合があります
- GitHub上に個別メモや価格情報が公開されるわけではありません
- APIなしなので、株価や移動平均は手入力です
- 本格的に同期したい場合は、今後Googleスプレッドシート、Supabase、Firebaseなどに保存する形へ拡張できます
