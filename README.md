# japan-stock-portal

スマホ中心で使う、日本株投資用の個人ポータルサイトです。

## できること

- 銘柄コードを入力して主要サイトへのリンクを自動生成
  - 株探
  - Yahoo!ファイナンス
  - TradingView
  - TDnet
  - EDINET
- テーマ別ウォッチリストを表示
  - 防衛
  - データセンター
  - 半導体
  - AI/SaaS
  - 資源
  - 電力
  - 蓄電池
  - IPO
- サイト上から銘柄を追加・編集・削除
- 銘柄ごとに以下を保存
  - ステータス：保有 / 監視 / 候補
  - エントリー価格
  - 損切りライン
  - 利確ライン
  - 決算日
  - 銘柄メモ
- エントリー価格、損切りライン、利確ラインから簡易R:Rを自動表示
- 投資メモをブラウザ内に保存
  - 今日の強いテーマ
  - 気になる銘柄
  - 決算跨ぎ候補
  - 損切りライン
  - 利確ライン
- TradingViewウィジェットを後から埋め込めるスペースを用意

## GitHub Pagesで公開する手順

1. GitHubでこのリポジトリを開く
2. `Settings` を開く
3. 左メニューの `Pages` を開く
4. `Build and deployment` の `Source` を `Deploy from a branch` にする
5. `Branch` を `main` にする
6. フォルダは `/root` を選ぶ
7. `Save` を押す
8. 数十秒〜数分待つと、公開URLが表示されます

スマホで操作する場合は、GitHubアプリよりもChromeでGitHubを開き、ブラウザの「PC版サイト」を使うと設定しやすいです。

## 銘柄を追加・編集・削除する方法

サイト上の `銘柄追加・編集` から入力できます。

入力できる項目：

- 銘柄コード
- 銘柄名
- テーマ
- ステータス
- エントリー価格
- 損切りライン
- 利確ライン
- 決算日
- 銘柄メモ

既存銘柄を編集する場合は、銘柄カード下部の `編集` を押してください。
削除する場合は、銘柄カード下部の `削除` を押してください。

## データ保存について

銘柄データと投資メモは、入力したスマホやブラウザの `localStorage` に保存されます。

注意点：

- 他の端末には同期されません
- ブラウザのデータを削除するとメモも消える場合があります
- GitHub上に個別メモや価格情報が公開されるわけではありません
- 本格的に同期したい場合は、今後Googleスプレッドシート、Supabase、Firebaseなどに保存する形へ拡張できます

## 初期銘柄をコード側で編集する方法

初期表示される銘柄は `script.js` の上部にある `defaultStocks` で管理しています。

例：

```javascript
{ id: "7011", code: "7011", name: "三菱重工業", theme: "防衛", status: "保有", entryPrice: "", stopLoss: "", targetPrice: "", earningsDate: "", memo: "防衛・宇宙・原子力も含む大型本命候補。" }
```

ただし、普段の銘柄追加・編集はサイト画面上から行う方が簡単です。
