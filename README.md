# japan-stock-portal

スマホ中心で使う、日本株投資用の個人ポータルサイトです。

## できること

- 銘柄コードを入力して主要サイトへのリンクを自動生成
  - 株探
  - Yahoo!ファイナンス
  - TradingView
  - TDnet
  - EDINET
- テーマ別ウォッチリストを表示
  - 防衛
  - データセンター
  - 半導体
  - AI/SaaS
  - 資源
  - 電力
  - 蓄電池
  - IPO
- 各銘柄のステータスを表示
  - 保有
  - 監視
  - 候補
- 投資メモをブラウザ内に保存
  - 今日の強いテーマ
  - 気になる銘柄
  - 決算跨ぎ候補
  - 損切りライン
  - 利確ライン
- TradingViewウィジェットを後から埋め込めるスペースを用意

## ファイル構成

```text
japan-stock-portal/
├── index.html   # 画面の骨組み
├── style.css    # デザイン
├── script.js    # 銘柄リンク生成、ウォッチリスト、メモ保存
└── README.md    # この説明書
```

## GitHub Pagesで公開する手順

1. GitHubでこのリポジトリを開く
2. `Settings` を開く
3. 左メニューの `Pages` を開く
4. `Build and deployment` の `Source` を `Deploy from a branch` にする
5. `Branch` を `main` にする
6. フォルダは `/root` を選ぶ
7. `Save` を押す
8. 数十秒〜数分待つと、公開URLが表示されます

スマホで操作する場合は、GitHubアプリよりもChromeでGitHubを開き、ブラウザの「PC版サイト」を使うと設定しやすいです。

## 銘柄を追加・編集する方法

`script.js` の上の方にある `stocks` を編集します。

例：

```javascript
{ code: "7011", name: "三菱重工業", theme: "防衛", status: "保有", memo: "防衛・宇宙・原子力も含む大型本命候補。" }
```

新しく追加するときは、同じ形で1行追加します。

```javascript
{ code: "XXXX", name: "銘柄名", theme: "テーマ名", status: "監視", memo: "メモを書く。" }
```

## テーマを追加する方法

`script.js` の `themes` にテーマ名を追加します。

```javascript
const themes = ["すべて", "防衛", "データセンター", "半導体", "AI/SaaS", "資源", "電力", "蓄電池", "IPO"];
```

そのうえで、`stocks` の各銘柄にも同じテーマ名を入れてください。

## 投資メモについて

投資メモは、入力したスマホやブラウザの `localStorage` に保存されます。

注意点：

- 他の端末には同期されません
- ブラウザのデータを削除するとメモも消える場合があります
- 本格的に保存したい場合は、今後GoogleスプレッドシートやSupabaseなどに保存する形へ拡張できます
